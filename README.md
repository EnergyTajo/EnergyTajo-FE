# EnergyTajo-FE

BDIA Devton 2024 공모전 - 프론트엔드
https://energytajo.site (모바일 접속 필수 or PC 접속 -> 개발자 모드)
https://www.energytajo.site

## 프로젝트 개요

### 주제

자가 발전 자전거를 이용한 친환경 전력 생산 및 지속 가능한 도시 개발 프로젝트

### 목표

- Figma UI 디자인 기반 JS 및 CSS 구현
- 웹앱 반응형 App 제작
- 포스트맨 데이터 바탕 API 백엔드 연동

### 개발 언어 및 환경

- 개발 언어 : React
- 개발 환경 : VisualCode
